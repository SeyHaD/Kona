# Kona
WordPress theme

## Install WordPress

Download the latest version of WordPress:
````
wget http://wordpress.org/latest.zip
````

Copy the files from `wordpress/*` to `./` and clean:
````
cp -R wordpress/* ./
rm -rif wordpress
rm -rif latest.zip
````
